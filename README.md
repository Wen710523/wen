# AI Assistant Welcome Page

A beautiful and interactive welcome page for an AI assistant with modern UI design and dynamic effects.

## Features

### Visual Effects
- **Dynamic Gradient Background** - Flowing animation with deep blue, purple, and pink colors (15s loop)
- **Floating Particles** - 20 animated particles floating in the background
- **Glassmorphism Design** - Modern frosted glass card effect
- **Smooth Animations** - Fade-in, slide-up, and hover effects

### Interactive Features
- **Real-time Clock** - Live time display that updates every second
- **Smart Greeting** - Auto-switches greetings based on time of day
- **Typewriter Effect** - Dynamic text with blinking cursor showing welcome messages
- **Search Box** - Functional search with Enter key and button support
- **Statistics Panel** - Animated counters for services, users, and accuracy
- **FAQ Section** - Expandable/collapsible frequently asked questions
- **Login Modal** - Complete login form with validation

### Technical Stack
- HTML5
- CSS3 (with animations and glassmorphism)
- Vanilla JavaScript (ES5 compatible)
- Node.js / PowerShell server

## File Structure

```
.
├── index.html          # Main page structure
├── styles.css          # Complete styling with animations
├── script.js           # Interactive logic and effects
├── server.js           # Node.js server script
└── start-server.ps1    # PowerShell server launcher
```

## Getting Started

### Option 1: PowerShell Server
```powershell
powershell -ExecutionPolicy Bypass -File start-server.ps1
```
Then open http://localhost:9000

### Option 2: Node.js Server
```bash
node server.js
```
Then open http://localhost:3000

### Option 3: Direct Open
Simply open `index.html` in your browser.
